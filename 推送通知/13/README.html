陀螺仪(了解)

一.陀螺仪的介绍

  • 陀螺仪主要用于检测设备的旋转
  • 应用场景:游戏中
  • 乔帮主演示陀螺仪视频

二.陀螺仪信息的获取

  • 通过CoreMotion框架来获取
  • 两种方式来获取
    • push方式
      • 创建运动管理者
      • 判断陀螺仪是否可用
      • 设置采样间隔
      • 开始采样
    • pull方式
      • 创建运动管理
      • 判断陀螺仪是否可用
      • 开始采样
      • 在需要时通过运动管理获取数值
  • push方式的代码实现
    // 1.判断陀螺仪是否可用
    if (!self.mgr.gyroAvailable) {
        NSLog(@"陀螺仪不可用");
        return;
    }

    // 2.设置采样间隔
    self.mgr.gyroUpdateInterval = 1.0;

    // 3.开始采样
    [self.mgr startGyroUpdatesToQueue:[NSOperationQueue mainQueue] withHandler:^(CMGyroData *gyroData, NSError *error) {
        if (error) return;

        // 4.获取陀螺仪信息
        CMRotationRate rate = gyroData.rotationRate;
        NSLog(@"x:%f y:%f z:%f", rate.x, rate.y, rate.z);
    }];
  • pull方式代码的实现
    // 1.判断陀螺仪是否可用
    if (!self.mgr.gyroAvailable) {
        NSLog(@"陀螺仪不可用");
        return;
    }

    // 2.开始采样
    [self.mgr startGyroUpdates];

    // 在需要的时候,主动获取.获取陀螺仪信息
    CMRotationRate rate = self.mgr.gyroData.rotationRate;
    NSLog(@"x:%f y:%f z:%f", rate.x, rate.y, rate.z);
